Mac 制作u盘启动盘

  • mac
Mac
准备工作

1.空U盘一枚。8G及以上

2.InstallESD.dmg

3.MacBook一坨

步骤：
  1. 我们打开系统自带磁盘工具。先把准备好的U盘抹除数据初始化。

  1. 分区时候注意一下分区选择GUID,如下图。

  1. U盘准备好以后，我们开始制作。

  1. 恢复完成以后，只需要再拷贝3个文件即可。

恢复完成以后会弹出来如下图这个框框，我们需要进入System/Installation（请原谅图片中的手误~）

进入文件夹后，我们删除Packages替身，然后将准备好的 OS X Install ESD 中的Packages文件夹拷贝到这里

拷贝完成后，我们回到OS X Base System根目录（也就是U盘根目录）将OSX Install ESD中的 BaseSystem.chunklist和BaseSystem.dmg复制过来。

至此启动盘已经制作ok，重启电脑，猛按option键，选择u盘，然后狂点下一步。系统就做好了。

ps：说一下提取InstallESD.dmg文件。怎么提取呢。

首先，在app store下载系统，在finer，应用程序中，右键，现实包含内容，Contents文件夹，SharedSupport文件夹，InstallESD.dmg。

其次，讲完了。

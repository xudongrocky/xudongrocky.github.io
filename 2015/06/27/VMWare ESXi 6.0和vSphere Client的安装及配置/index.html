VMWare ESXi 6.0和vSphere Client的安装及配置

服务器机型：DELL T430

由于公司要求，安装VMWare ESXi6.0 然后安装Windows Server 2008 R2 Sta 以及 Windows Server 2012 R2 Sta 。

​准备：

免费VMWare请下载：https://my.vmware.com/cn/web/vmware/evalcenter?p=free-esxi6

注册帐号，下载VMWare ESXi6.0 镜像文件。写入U盘或者光盘（需要刻录机RW）。

ok,开机F10进入BIOS，设置U盘启动。

至此结束。
